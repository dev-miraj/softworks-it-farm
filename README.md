# SOFTWORKS IT FARM

  A premium tech studio full-stack digital platform built with React, TypeScript, Node.js, and PostgreSQL.

  ## Features
  - Public-facing website (Home, About, Services, Portfolio, SaaS Products, Blog, Contact)
  - Admin CMS Panel  
  - HR Management System (Employees, Attendance, Leave Requests, Payroll)

  ## Tech Stack
  - Frontend: React + Vite + TypeScript + Tailwind CSS
  - Backend: Express + Node.js
  - Database: PostgreSQL + Drizzle ORM
  - Animations: GSAP + ScrollTrigger

  ## Setup
  ```bash
  pnpm install
  pnpm dev
  ```
